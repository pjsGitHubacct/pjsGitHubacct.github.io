# Agentforce - Sales Cloud Lead Qualification Agent

This repository contains a Salesforce **Agentforce** implementation for automating inbound lead qualification in **Sales Cloud**.

## Features
- Autonomous lead qualification
- CRM data grounding using Salesforce objects
- Automatic lead status updates
- Handoff to sales reps when qualification criteria are met

## Architecture
- Salesforce Agentforce
- Salescloud (Lead, Account, Activity objects)
- Prompt Templates
- Salesforce Flow for actions

# Prerequisites
- Salesforce org with Sales Cloud enabled
- Agentforce enabled
- Data Cloud (recommended for grounding)
- System Administrator access

## Setup Steps
1. Enable Agentforce in Setup
2. Create a new Agent using Agent Builder
3. Configure Topics and Instructions
4. Asssign Salesforce Flows as Actions
5. Test using Reasoning Logs
6. Deploy to Sales Users / public website created using github pages.
